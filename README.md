# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to understand unemployment patterns and the impact of COVID-19.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Unemployment in India Dataset

## Features
- Data Cleaning
- Exploratory Data Analysis
- State-wise Unemployment Analysis
- Urban vs Rural Comparison
- Monthly Trend Analysis
- COVID-19 Impact Analysis
- Correlation Heatmap

## How to Run
1. Clone the repository.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Open `Unemployment_Analysis.ipynb` in Jupyter Notebook.
4. Run all cells.
